# Stylized Junk Boat Render Prompt

## Prompt

Create a stylized illustrated render of the Junk Boat playground at Lincoln Square Park in Oakland Chinatown, using the uploaded real reference photos for structure and the uploaded style reference image for mood and composition.

The image should feel like a bright daytime museum exhibit centerpiece: playful but polished, with simplified geometric forms, warm storybook color, a floating-island or low-poly architectural illustration feel, and enough realism to preserve the identity of the actual playground. Make it inviting, colorful, and curated rather than toy-like.

Preserve the recognizable Junk Boat structure: a red ship-like hull, slide, stairs, railings, tall sail or mast structures, circular openings, and the surrounding playground setting. Use clean shapes, crisp edges, soft shadows, and a slightly elevated three-quarter view that makes the boat readable as a single exhibit object.

Do not make the image photorealistic. Do not make it dark, futuristic, cyberpunk, or game-like. Do not include people, characters, text, logos, labels, or UI elements.

Final output should be a high-resolution PNG with a wide composition suitable for a website hero or interactive centerpiece. Save the finished file as:

`public/images/junk-boat-stylized-render.png`
